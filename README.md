# dotfiles

## how to use

`git clone https://github.com/87YLu/dotfiles.git`

`ln -s ~/dotfiles/nvim ~/.config/nvim`

## nvim

### Prerequisites

| dependence | download             |
| ---------- | -------------------- |
| ripgrep    | brew install ripgrep |
| wget       | brew install wget    |
| fd-find    | npm i fd-find -g     |
